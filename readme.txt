1.物流快递管理系统分为客户端与服务器两个部分。客户端的启动器
为client文件夹中的client_launcher.jar，服务器的启动器为server
文件夹中的server_launcher.jar。

2.服务器局域网的ip地址可以在server和client文件夹中IPConfig.txt中修改。

3.在启动程序之前请务必保证机器安装了jre或者jdk。

4.要删除数据库请删除server文件夹目录下的LogisticsManageSystemDataBase文件夹
以及derby.log文件
